🔄 Automation & Updates

This site is completely automated. Data updates run every day at 2 AM UTC via GitHub Actions with zero manual intervention.

How It Works

  1. GitHub Actions wakes up on schedule
  2. Fetches fresh data from 7 government APIs
  3. Validates and reconciles data
  4. Allocates national impacts to states
  5. Updates SQLite time-series database
  6. Exports JSON for this visualization
  7. Commits changes to repository
  8. Site auto-deploys to GitHub Pages
  9. No human work required
